Tips for Using Icons

Case Insensitive: You can use "CV", "cv", or "Cv" - they all work the same.

Partial Matching: The system is smart enough to match "Mobile App" to the üì± icon.

Fallback: If no match is found, a random icon will be assigned automatically.

Easy Updates: Just change the text in your CSV file to get different icons!
